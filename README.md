# Sakha Product API

A full-featured product management REST API built with **FastAPI** and **SurrealDB**. Covers the complete e-commerce lifecycle: products, variants, inventory, cart, orders, reviews, coupons, analytics, and more.

## Documentation Directory

We have broken down the documentation into focused guides to make it easier to digest:

1. **[Setup & Installation Guide](backend/docs/setup_guide.md)**
   Everything you need to know about getting your local environment up and running, starting the services, and seeding the database.

2. **[API Reference & Integration](backend/docs/api_reference.md)**
   Extensive documentation covering all capabilities of the REST API. Contains details on authentication workflows, route configurations, data shapes, error codes, and the entire backend functionality ecosystem (from Auth & Products to Cart & Orders).

3. **[Architecture & Testing](backend/docs/architecture_and_tests.md)**
   Information regarding our technical stack, detailed project structure map, and instructions on how to navigate and execute the test suites securely utilizing the mocked environment.
